# 🎁 绩点盲盒抽奖

简单的绩点盲盒抽奖网页，部署在 Render。

## 功能

- 输入精网号登录
- 点击盲盒抽奖（1-20 绩点随机）
- 查看抽奖记录

## 抽奖概率

| 绩点 | 概率 |
|------|------|
| 1 | 15% |
| 2 | 15% |
| 3 | 12% |
| 4-5 | 10% |
| 6 | 8% |
| 7 | 7% |
| 8 | 6% |
| 9 | 5% |
| 10 | 4% |
| 12 | 3% |
| 15 | 2% |
| 20 | 1% |

## 本地运行

```bash
npm install
npm start
```

## 部署

1. 推送到 GitHub
2. 在 Render 创建 Web Service
3. 设置启动命令: `npm start`
